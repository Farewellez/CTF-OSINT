# 01_flag_format
## 問題文
CTFでは、「答え」をFlag（フラグ）と呼びます。これから先、すべての調査事項はFlagとして提出する必要がありますから、まずは答え方を練習してみましょう。<br>
さて、添付ファイル（briefing_flag_format.jpg）に写っているタワーの名前を、英語表記 で解答してください。<br>
Flag形式は SWIMMER{xxxx Tower} です。もし 京都タワー が答えである場合、Flagは SWIMMER{Kyoto Tower} となります。<br>
なお、各調査事項ごとにこのような「Flag形式」という例示を行いますので、提出時には必ず確認してください。<br>

In CTFs, the "answer" is called a flag. From now on, every investigation item must be submitted as a flag, so let's practice how to answer.<br>

Answer the name of the tower shown in the attached file (briefing_flag_format.jpg) in English.<br>
The flag format is SWIMMER{xxxx Tower}. If the answer were Kyoto Tower, the flag should be SWIMMER{Kyoto Tower}.<br>
Each investigation item will include an example "flag format" like this, so be sure to check it when submitting.<br>

## 解法 
Here we can just open Google Lens in Google search and we will immediately know that it is Tokyo Tower. so the flag is Tokyo Tower<br>
**Flag: SWIMMER{Tokyo Tower}**
